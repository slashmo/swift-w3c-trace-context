Trace​Context

An implementation of W3C TraceContext, combining TraceParent and TraceState.

Trace​Parent

Represents the incoming request in a tracing system in a common format, understood by all vendors.

Trace​State

Provides additional vendor-specific trace identification information across different distributed tracing systems and is a companion header for the traceparent field.
