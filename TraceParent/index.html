Structure Trace​Parent 

public struct TraceParent

Represents the incoming request in a tracing system in a common format, understood by all vendors.

Example raw value: 00-0af7651916cd43dd8448eb211c80319c-b7ad6b7169203331-01

trace​ID 

let traceID: String

The ID of the whole trace forest, used to uniquely identify a distributed trace through a system.

W3C TraceContext: trace-id

parent​ID 

let parentID: String

The ID of the incoming request as known by the caller (in some tracing systems, this is known as the span-id, where a span is the execution of a client request).

W3C TraceContext: parent-id

trace​Flags 

let traceFlags: String

An 8-bit field that controls tracing flags such as sampling, trace level, etc.

W3C TraceContext: trace-flags

sampled 

var sampled: Bool

When true, the least significant bit (right-most), denotes that the caller may have recorded trace data. When false, the caller did not record trace data out-of-band.

W3C TraceContext: Sampled flag
