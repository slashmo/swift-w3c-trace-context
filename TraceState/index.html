public struct TraceState

Provides additional vendor-specific trace identification information across different distributed tracing systems and is a companion header for the traceparent field.

Example raw value: rojo=00f067aa0ba902b7,congo=t61rcWkgMzE

public init?(rawValue: String)

Initialize a TraceState from an HTTP header value. Fails if the value cannot be parsed.

When receiving multiple header fields for tracestate, rawValue should be a joined, comma-separated String of all values according to HTTP RFC 7230: Field Order.
